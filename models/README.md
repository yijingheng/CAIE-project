# Models

This directory contains scripts and information for downloading pre-trained models.

## Pre-trained Models
- `yolov8_led.pt`: YOLOv8 model fine-tuned for LED detection
- `lux_model.pth`: PyTorch model for lux sensor data prediction

## Downloading Models
Run `python download_models.py` to download the required models.

## Model Sources
- YOLOv8: https://github.com/ultralytics/ultralytics
- Lux models: Trained on project-specific data

Note: Models are not included in the repository due to size constraints.
